SEMANA 4 - 2

  1. Box Model

    2. Esta definida en el navegador como una caja rectangular, esta es la forma en que se representan todos los elementos.

    Box Model

  2. Padding

    3. Es una separación o espacio interior que existe entre el contenido y el borde de la caja.

    Ejemplo:
     padding-top: 10px;
    padding-right: 20px;
    padding-bottom: 25px;
    padding-left: 5px;
    Los 4 valores son igual a esto:
    padding: 10px 20px 25px 5px;
    padding: 80px; Aplica a los cuatro lados
    padding: 10px 20px; vertical | horizontal
    padding: 10px 20px 30px; top | horizontal | bottom

    Padding Box

  3. Border

    4. El borde es la línea que rodea la caja, es como la frontera que rodea al elemento, esta se utiliza para darle una apariencia estética a la caja.

    Ejemplo:
     border-width: 5px 10px 15px 20px;
    border-style: solid dashed dotted double;
    border-color: red green blue brown;

    Border Box

  4. Margin

    5. El margen es la última área del Box Model y se puede ver como una separación invisible o transparente que ayuda a separar un elemento de otro.

    Ejemplo:
     margin-top: 10px;
    margin-right: 20px;
    margin-bottom: 25px;
    margin-left: 5px;
    Los 4 valores son igual a esto:
    margin: 10px 20px 25px 5px;
    margin: 10px; Aplica a los cuatro lados
    margin: 10px 20px; vertical | horizontal
    margin: 10px 20px 30px; top | horizontal | bottom

    Margin Box

  5. Selector Universal

    6. Selecciona todos los elementos en un documento.

    Ejemplo:
     * o *|* { propiedades de estilo } - selecciona todos los elementos:
     *{ box-sizing: border-box;}

  6. Margenes Negativos

    7. Son valores negativos que se establecen en la propiedad de los margenes de un elemento.

    Ejemplo:
     margin-top: -10px;
    margin-right: -20px;
    margin-bottom: -25px;
    margin-left: -5px;
    Los 4 valores son igual a esto:
    margin: -10px -20px -25px -5px;
    margin: -10px; Aplica a los cuatro lados
    margin: -10px -20px; vertical | horizontal
    margin: -10px -20px -30px; top | horizontal | bottom

  7. Colapsado de Margenes Verticales

    8. Son elementos de bloque de HTML que tienen márgenes verticales que se chocan entre sí, estos dos márgenes colapsan en uno o se fusionan entre sí, y aquí dominará el de mayor valor.

    Ejemplo:
     .box-1 {
    margin-bottom: 100px; ⬅
    padding: 20px;
    border: solid 5px red;
    }
    .box-2 {
    margin-top: 40px; ⬅
    padding: 20px;
    border: solid 5px blue;
    }
    span {
    font-weight: bold;
    }
     El que domina es el box-1
